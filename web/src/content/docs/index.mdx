## Quick Start

```html
<script
  type="module"
  src="https://cdn.jsdelivr.net/npm/ds-one@0.1.8/dist/index.esm.js"
></script>
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/ds-one@0.1.8/DS1/1-root/one.css"
/>

<button-v1 variant="primary">Hello World</button-v1>
<text-v1 variant="heading">Welcome to DS one</text-v1>
```

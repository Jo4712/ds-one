DS one Link Component

Basic Link (External)

This is a basic external link that opens in a new tab:

<link-v1 href="https://example.com"></link-v1>

Link with Custom Text

This link uses a custom translation key:

<link-v1 href="https://github.com" key="getStarted"></link-v1>

Internal Link

This link opens in the same tab (internal navigation):

<link-v1 href="/about" external="false"></link-v1>

Link with Custom Target and Rel

This link has custom target and rel attributes:

<link-v1 href="https://example.com" target="_blank" rel="noopener noreferrer nofollow"></link-v1>

Link without href (Text Only)

When no href is provided, it renders as plain text:

<link-v1></link-v1>

Link with Default Value

This link uses a custom default value when the translation key is not found:

<link-v1 href="https://example.com" key="nonexistent" default-value="Custom Link Text"></link-v1>

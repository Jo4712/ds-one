DS one - CDN Support Coming Soon

⚠️ CDN Support Not Available Yet

The current version (0.1.0-beta) does not support CDN usage due to external dependencies. CDN support will be added in a future release.

See GitHub Issue #1 for progress updates.

Alternative: Install via Bun/NPM

For now, please install DS one using a package manager:

# Using bun (recommended)
bun add ds-one@beta

# Using npm
npm install ds-one@beta

Usage After Installation

<!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" href="node_modules/ds-one/DS/1 Root/screen.css" />
    <script type="module" src="../DS/2 Core/app-v1.ts"></script>
    <script type="module" src="../DS/2 Core/button-v1.ts"></script>
    <script type="module" src="../DS/2 Core/text-v1.ts"></script>
  </head>
  <body>
    <app-v1 type="default">
      <text-v1 variant="heading">Welcome to DS one</text-v1>
      <button-v1 variant="primary">Get Started</button-v1>
    </app-v1>
  </body>
</html>
